## **Flujo de trabajo de GIT**
### Para añadir un fichero a nuestro repositorio remoto de [GitHub](https://pages.github.com/) tenemos que seguir los siguientes pasos. 

```
git add fichero
git commit -m " Comentario sobre el commit"
git push

```
